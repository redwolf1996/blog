git删除分支

2023-02-13

  1. 使用git branch -a查看左右分支，分支形如 『remotes/origin/<branch_name>』 的的是远程分支，其他的是本地分支，如果要删除某些分支，请先使用git checkout切换到这些以外的分支

  2. 使用git banch -d <branch_name>删除本地分支，如果无法删除，可以使用git branch -D <branch_name>强制删除

  3. 使用git push origin -d <branch_name>删除远程分支
